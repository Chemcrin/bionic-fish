# 仿生鱼工程

> 面向水下任务演示的仿生机器鱼项目，以稳定性和可控性为核心目标。

---

## 项目简介

本项目设计并制作一只以**稳定游动、可控沉浮、精准方向控制**为优先目标的仿生机器鱼，用于水下任务演示或比赛。首版采用线控方案验证原型，后续阶段再引入自主决策与视觉识别。

### 核心能力

| 能力 | 说明 |
|:---:|------|
| 基础推进 | 尾部舵机驱动尾鳍周期摆动，提供前进动力 |
| 方向控制 | 尾鳍、背鳍、臀鳍配合完成左转、右转及航向修正 |
| 可控沉浮 | 中性浮力 + 鳍面控制，实现上浮、下潜和定深 |
| 任务打卡 | 接近不同坐标点并完成触发/停留/识别 |
| 水下安全 | 主控、舵机、电池、线缆区域可靠防水 |

## 技术路线

### 阶段 A：可靠线控原型（当前阶段）

```
主控：STM32F103ZET6
舵机：DS3218MG 防水舵机 × 3~4
通信：UART 串口线控
外壳：PETG 3D 打印
姿态传感器：JY61P
深度传感器：MS5837
电源：2S 航模锂电池 7.4V + 分级稳压
```

### 阶段 B：自主增强版本（规划中）

- 更强算力 MCU/协处理器（ESP32-S3）
- 目标识别与自动寻点（传感器方案）
- 深度闭环、姿态闭环、任务状态机

## 系统架构

```
操作者/上位机 ──线控指令──→ STM32 主控
                              │
              ┌───────────────┼───────────────┐
              ▼               ▼               ▼
         舵机驱动        姿态/深度传感器     漏水检测
              │               │               │
    ┌─────────┼─────────┐     │               │
    ▼         ▼         ▼     │               │
 尾鳍推进  背鳍控制  臀鳍控制  │               │
                               ▼               ▼
                        电源与保护模块     可选视觉模块

外壳：3D 打印鱼体 → 密封电控舱 → 可调配重/浮力模块
```

---

## 沉浮控制方案

采用 **中性浮力 + 鳍面控制升降** 方案：

1. 通过配重块和浮力材料将鱼体调至接近中性浮力
2. 背鳍/臀鳍调整角度产生升力或下压力，类似飞机俯仰舵面
3. JY61P 姿态传感器实时反馈，实现闭环姿态稳定

## 关键风险

| 风险 | 应对 |
|------|------|
| 水下通信不稳定 | 首版 UART 串口线控，验证短距离稳定性 |
| 防水失败 | 分阶段浸水测试，电控舱独立密封 |
| 重心与浮心不匹配 | 可调配重，静水调平后再动态测试 |
| 舵机力矩不足 | 预留更大扭矩舵机安装空间 |
| 沉浮控制不足 | 首版调中性浮力，必要时升级主动浮力舱 |

## 验收标准

- 静置浸水 30 分钟，电控舱和电池舱无明显进水
- 入水前自检通过，舵机可回中，通信稳定
- 静水中鱼体保持基本水平，不持续侧翻
- 尾鳍摆动后能连续前进，舵机无明显过热
- 能完成可控左转、右转和航向修正
- 能完成上浮、下潜或不同深度目标接近
- 能按比赛规则完成至少一次目标点打卡
