# 仿生鱼工程

> 面向水下任务演示的仿生机器鱼项目，以稳定性和可控性为核心目标。

---

## 项目简介

本项目设计并制作一只以**稳定游动、可控沉浮、精准方向控制**为优先目标的**仿生虎鲸**（体长 50-60cm），用于水下任务演示或比赛。

| 推进 | 转向 | 姿态 | 通信 |
|:----:|:----:|:----:|:----:|
| 防水电机 + 曲柄摇杆 | 舵机 + 齿轮组联动背鳍与尾腰 | 左右腹鳍独立舵机 | ESP32 无线中继 + 背鳍天线 |

### 核心能力

| 能力 | 说明 |
|:---:|------|
| 基础推进 | 防水电机驱动尾鳍上下摆动（曲柄摇杆机构） |
| 方向控制 | 背鳍舵机通过齿轮组联动尾段扭腰实现转向 |
| 姿态辅助 | 左右腹鳍独立控制，辅助横滚修正和沉浮微调 |
| 可控沉浮 | 中性浮力 + 鳍面控制，实现上浮、下潜和定深 |
| 任务打卡 | 接近不同坐标点并完成触发/停留/识别 |
| 水下安全 | 主控、舵机、电机、电池、线缆区域可靠防水 |

## 技术路线

### 阶段 A：可靠无线原型（当前阶段）

```
尺寸：体长 50~60cm（仿生虎鲸）
推进：防水电机 × 1 + 曲柄摇杆 → 尾鳍上下摆动
转向：舵机 × 1 + 齿轮组 → 联动背鳍与尾段扭腰
姿态：舵机 × 2 → 左右腹鳍独立控制
主控：STM32F103ZET6（运动控制 + 传感器）
通信中继：ESP32（带 IPEX 接口，2.4G 无线）
天线：IPEX 同轴线延伸至背鳍内部
外壳：PETG 3D 打印
上位机：遥控器/手机/PC
```

### 阶段 B：自主增强版本（规划中）

- 更强算力 MCU/协处理器（ESP32-S3）
- 目标识别与自动寻点（传感器方案）
- 深度闭环、姿态闭环、任务状态机

## 沉浮控制方案

采用 **中性浮力 + 鳍面控制升降** 方案：

1. 通过配重块和浮力材料将鱼体调至接近中性浮力
2. 腹鳍（主）和背鳍（辅）调整角度产生升力或下压力
3. JY61P 姿态传感器实时反馈，实现闭环姿态稳定

## 关键风险

| 风险 | 应对 |
|------|------|
| 水下 2.4G 衰减 | ESP32 中继 + 背鳍天线；必要时降级为浮标中继 |
| 防水电机进水 | 选用正规防水电机，轴封和接线处额外密封 |
| 曲柄摇杆卡滞 | 运动仿真验证，打印样件手动测试 |
| 齿轮组干涉/堵转 | 合理齿轮侧隙，预留调节余量 |
| 重心与浮心不匹配 | 可调配重，静水调平后再动态测试 |
| 沉浮控制不足 | 首版调中性浮力，必要时升级主动浮力舱 |

## 验收标准

- 静置浸水 30 分钟，电控舱和电池舱无明显进水
- 入水前自检通过，电机/舵机可回中，通信稳定
- 静水中鱼体保持基本水平，不持续侧翻
- 尾鳍上下摆动后能连续前进，电机无明显示过热
- 背鳍联动尾腰，能完成可控左转、右转和航向修正
- 左右腹鳍可独立控制，辅助横滚修正和沉浮微调
- 能完成上浮、下潜或不同深度目标接近
- 能按比赛规则完成至少一次目标点打卡
